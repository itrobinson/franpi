# Get the essentials
To get started you'll need some hardware:

- a [Raspberry Pi](https://www.raspberrypi.com/)
- loudspeakers or headphones
- an Ethernet cable, unless you have WiFi
- a Secure Digital (SD) card
- a SD card reader
- a power supply
